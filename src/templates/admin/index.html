Project Description

This project demonstrates a realistic vulnerable web application combined with an automated exploit framework for educational and security training purposes.

The application intentionally contains two(2) of OWASP Top 10 vulnerabilities and other Vulnerabilities like SQL Injection, XSS, Broken Authentication, IDOR, Path Traversal and File Upload vulnerabilities.

Automated Python exploit scripts are used to demonstrate practical exploitation techniques and vulnerability analysis.

The platform helps demonstrate how common web vulnerabilities can be exploited in realistic attack scenarios.
